# 💰 PaisaKid - Digital Canteen Wallet

> **Micathon '26 Theme: Money Moves** > *Making money work better for the next generation of students.*

[cite_start]**PaisaKid** is a cashless, cardless, and phone-less "digital wallet" system designed specifically for school canteens in Pakistan[cite: 152, 172]. [cite_start]It bridges the gap for kids who are too young for phones and parents who want total visibility into daily spending[cite: 173].

---

## 🌟 The Core Problem
- **The Gap:** In Pakistan, young children don't carry smartphones. [cite_start]Physical cash leads to loss, theft, and a total lack of visibility for parents regarding nutrition and spending[cite: 154].
- [cite_start]**The Friction:** Montessori and Prep-level children cannot remember complex PINs or navigate traditional banking apps[cite: 176].

## 🚀 The Solution
### 👶 For the Student: Photo-Tap Authentication
Zero technology required. Students pay by simply tapping their own face from a randomized grid of four photos. It is instinctive for a 3-year-old and serves as a physical proof of presence.

### 🏪 For the Shopkeeper: Streamlined POS
A simple interface where the vendor enters a Student ID and verifies identity instantly.

### 👪 For the Parent: Real-time Visibility
[cite_start]A dashboard to set daily spending limits, top up balances (via JazzCash/EasyPaisa simulations), and receive itemized digital receipts[cite: 196].

---

## 🛠️ Technical Architecture
- [cite_start]**Backend:** Django (Python) leveraging a robust Closed-Loop Ledger system[cite: 226].
- [cite_start]**Frontend:** React for a responsive, fast-paced canteen experience[cite: 227].
- [cite_start]**Security:** Role-Based Access Control (RBAC) and atomic transaction handling to ensure data integrity[cite: 201].
- **Real-time Updates:** Optimized polling to ensure parents see "Money Moves" as they happen.
